## 🏗 Installer et utliser en local

Clonez le repo avec Git. Seuls Node.js (testé en v14.3.0) et Yarn (testé en 1.22.4) sont nécessaires.

Installez les dépendances:

```sh
npm install
```

Puis lancez le serveur de dev avec:

```sh
npm run dev
```
